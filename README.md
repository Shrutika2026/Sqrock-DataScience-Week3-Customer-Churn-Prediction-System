# Customer Churn Prediction System

## 📌 Project Overview
This project was developed as part of **Week 3: Data Science Task Project**.  
https://colab.research.google.com/drive/18qFUEZLQRmMMMndM548ierLZ2rX2lK_Z?usp=sharing
The objective is to build a machine learning model that predicts whether a customer is likely to leave a company (churn) based on given data.  
Customer churn prediction is widely used in industries like telecom, banking, and SaaS, making this project highly valuable for practical applications and resume building.

---

## 🎯 Objectives
- Analyze customer data
- Identify patterns behind customer churn
- Build a prediction model using machine learning
- Evaluate model performance and derive actionable insights

---

## 📂 Dataset
- **Telco Customer Churn Dataset** (available on Kaggle)
- Contains customer demographics, account information, and service usage details

---

## 🛠️ Core Tasks

### 1. Data Cleaning
- Handle missing values
- Encode categorical variables
- Remove irrelevant columns

### 2. Exploratory Data Analysis (EDA)
- Calculate overall churn rate
- Identify which customer groups churn more
- Study impact of features such as tenure, contract type, monthly charges

### 3. Data Visualization
- Bar charts for churn distribution
- Count plots for categorical features
- Correlation heatmap for numerical features

### 4. Machine Learning Model
- **Logistic Regression** (mandatory)
- **Decision Tree** (optional)

### 5. Model Evaluation
- Accuracy
- Confusion Matrix
- Precision & Recall

---

## 🚀 Advanced Features (Bonus)
- Compare multiple models
- Feature importance analysis
- Build a simple UI using Streamlit
- Predict churn for new user inputs
